phonepe_project Data Analysis & Visualization
This project focuses on analyzing and visualizing PhonePe Pulse transaction data to uncover valuable business insights related to digital payments, customer behavior, geographical trends, and transaction performance.
The project uses:
- Python for data extraction, preprocessing, analysis, and machine learning
- SQL concepts for structured data handling
- Pandas, Matplotlib, Seaborn for visualization
- Scikit-learn for machine learning models
- Streamlit for dashboard creation

The dataset is extracted from the official PhonePe Pulse GitHub repository.

Problem Statement:
With the increasing reliance on digital payment systems like PhonePe, understanding transaction dynamics, user engagement, and geographical trends is essential for improving services and targeting users effectively.

This project aims to:
- Analyze aggregated transaction data
- Visualize transaction trends
- Identify top-performing states and transaction categories
- Perform customer segmentation using clustering
- Predict transaction amounts using regression models

Technologies Used:
